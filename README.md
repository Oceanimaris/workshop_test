# Vanessa Quarto Website Template

This is a starter Quarto website structure based on a standard researcher website workflow.

## Included pages

- `index.qmd`
- `research.qmd`
- `publications.qmd`
- `projects.qmd`
- `cv.qmd`
- `news/2026.qmd`

## To use

1. Open a terminal in this folder
2. Run:

```bash
quarto preview
```

3. Replace placeholders:
   - `your.email@maine.edu`
   - `https://github.com/yourusername`
   - `images/headshot-placeholder.jpg`
   - `files/vanessa-mahan-cv.pdf`

## Suggested next edits

- add your actual headshot to `images/`
- add your CV PDF to `files/`
- update publication entries
- add project screenshots or figures
- connect the repo to GitHub Pages or Quarto Pub
